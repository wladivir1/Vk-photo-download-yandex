Чтобы использовать эту программу, вам понадобится установить дополнительные зависимости:

pip install requests tqdm

Кроме того, у вас должен быть действующий токен Яндекс.Диска, 
полученный на Полигоне Яндекс.Диска.

Программа запросит у вас id пользователя vk, токен Яндекс.Диска.

Пример запуска программы:

Введите id пользователя vk: 123456789
Введите токен с Полигона Яндекс.Диска: YOUR_YANDEX_DISK_TOKEN
Введите путь к папке на Я.Диске для сохранения фотографий (например, '/photos'): /my_photos
Сохранение фотографий: 100%|███████████████████████████| 5/5 [00:03<00:00,  1.38it/s]
Фотографии успешно сохранены.
Программа сохранит определенное количество фотографий максимального размера 
с профиля пользователя в папку на Яндекс.Диске, а также создаст JSON-файл с информацией о каждой фотографии.
